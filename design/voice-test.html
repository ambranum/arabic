Can respelling force Palestinian pronunciation out of an MSA voice?

In Levantine Arabic, ق is pronounced as a glottal stop in most words — but it's still written as ق. So the spelling itself is what drags any text-to-speech engine back into fuṣḥā. This app generates its own dialect text, which means we control what the engine is fed, independently of what you see on screen.

Each row below is the same word twice: as it's spelled, and as it's said. Both are spoken by Majed — the one MSA voice on this Mac. If the right-hand column sounds Palestinian, the fix is orthographic, not a model problem.
